# Crop Band ROI (Quick Guide)

This utility crops PNG slices generated by area-band runs (`axis_x/axis_y/axis_z/brain_slices`) into a smaller ROI.

Script: `functions/crop_band_roi.py`  
Config: `configs/crop_band_config.json`

## How cropping works
- The crop is applied with pixel slicing: `img[y0:y1, x0:x1]`.
- Bounding box format is normalized: `(x, y, w, h)` in `[0, 1]`.
- Priority:
1. `bbox_norm` (global for all axes, if set)
2. `bbox_norm_by_axis[axis_*]` (axis-specific)
3. Auto-detect from foreground (only if allowed)

## Recommended config pattern
Edit `configs/crop_band_config.json`:
- `band_root`:
  - Single subject path, or
  - Parent folder containing many subject folders.
- `bbox_norm_by_axis`:
  - Put known axis bbox values.
  - Keep unknown axes as `null`.
- `allow_auto_for_unset_axes`:
  - `false` (recommended): skip axes with no bbox.
  - `true`: auto-crop unset axes.
- `overwrite`:
  - `false`: keep existing crops.
  - `true`: always replace existing crops.

Example:
```json
"bbox_norm_by_axis": {
  "axis_x": null,
  "axis_y": [0.546875, 0.171875, 0.29166, 0.3072916],
  "axis_z": null
},
"allow_auto_for_unset_axes": false
```

## Run command
```powershell
python -m functions.crop_band_roi --config configs/crop_band_config.json
```

Optional overwrite from CLI:
```powershell
python -m functions.crop_band_roi --config configs/crop_band_config.json --overwrite
```

## Output location
For each processed axis:
- Input: `<subject>/axis_*/brain_slices/*.png`
- Output: `<subject>/axis_*/brain_slices_cropped/*.png`
- Manifest: `<subject>/crop_manifest.json`

## Troubleshooting
- Only one subject processed:
  - `band_root` is set to a single subject folder.
  - Set it to the parent (for example `...\area_band_output_multi`) to process all subjects.
- Crop too tall/full height:
  - That axis likely used auto-crop.
  - Add bbox for that axis or keep `allow_auto_for_unset_axes: false`.
- No `brain_slices_cropped` folder:
  - That subject/axis likely has no `brain_slices` input or was skipped by config rules.
